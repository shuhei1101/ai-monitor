# モック索引

4 つのビューから入る。

| ビュー | 説明 |
| ---------------------------- | ------------------------------------------------------------------------ |
| [画面（pages）](pages/) | 画面ごとに Issue 一覧 → 案リスト → 案の実画面 と辿る |
| [コンポーネント（components）](components/) | 共通コンポーネントの全案を 1 ページに並べたカタログ |
| [スタイル（styles）](styles/) | 画面全体のデザインスタイル。採用 1 案 + 候補案を画面で見比べる |
| [Issue 横断（issues）](issues/) | 1 Issue で複数画面を扱う場合に、その Issue から関連画面に最短で辿る |
