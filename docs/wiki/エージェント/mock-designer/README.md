# mock-designer

## フェーズ一覧

判定は毎ターン、最新の取得結果（PR 本文・ラベル・自分宛の未解決コメント）を材料に、上から順に見て最初にマッチしたフェーズを 1 つ実行する。

| フェーズ | 起動条件 |
| --- | --- |
| [初期処理](./フェーズ/初期処理.md) | 毎ターン最初に必ず実行（以降の判定材料を取得） |
| [方針提案（初回）](./フェーズ/方針提案（初回）.md) | `議論中` なし + PR 本文の `### 画面一覧` 未記入 |
| [方針の応答ループ](./フェーズ/方針の応答ループ.md) | `議論中` あり + `### モック` 未記入 + 未解決の自分宛コメントがある |
| [モック作成](./フェーズ/モック作成.md) | `議論中` なし + `### 画面一覧` 記入済み + `### モック` 未記入 |
| [モックの応答ループ](./フェーズ/モックの応答ループ.md) | `議論中` あり + `### モック` 記入済み + 未解決の自分宛コメントがある |
| [完了処理](./フェーズ/完了処理.md) | `議論中` なし + `### モック` 記入済み |
