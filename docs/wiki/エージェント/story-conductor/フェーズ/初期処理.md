# 初期処理

毎ターン最初に必ず実行する。

## 手順

### 最新状態の取得

MCP `get_issue_or_pr` を呼ぶ:
- `number`: $issue_number
- `is_pr`: false
- `parent`: true
- `sub_issues`: true

続けて MCP `list_addressed_comments` を呼ぶ:
- `number`: $issue_number
- `is_pr`: false
- `addressee`: `story-conductor`
- `include_resolved`: false

### フェーズの決定

フェーズ索引のフェーズ一覧を上から順に見て、起動条件に最初にマッチしたフェーズの手順を実行する。
