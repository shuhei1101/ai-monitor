# single-scenario-writer

## フェーズ一覧

判定は毎ターン、最新の取得結果（PR 本文・ラベル・自分宛のコメント）を材料に、上から順に見て最初にマッチしたフェーズを 1 つ実行する。

| フェーズ | 起動条件 |
| --- | --- |
| [初期処理](./フェーズ/初期処理.md) | 毎ターン最初に必ず実行（以降の判定材料を取得） |
| [シナリオ作成（初回）](./フェーズ/シナリオ作成（初回）.md) | `議論中` なし + 自身の投稿コメントなし（Resolved 済み含めて 0 件） |
| [応答ループ](./フェーズ/応答ループ.md) | `議論中` あり + 未解決の自分宛コメントがある |
| [完了処理](./フェーズ/完了処理.md) | `議論中` なし + 自身の投稿コメントあり |
